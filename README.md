#Turbine app aggregates all the payment related apis and and process it further
##Project Setup Summary

###To setup the project on local machine
- clone the git repo from bitbucket: https://bitbucket.org/zee5in/turbine/src/master/
- go to project directory: cd Turbine
- build project: mvn clean install
- run project: mvn spring-boot:run
- use swagger to see apis using url: http://localhost:8080/turbine/swagger-ui/index.html#/
